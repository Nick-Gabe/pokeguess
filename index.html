Como Funciona

Adivinhe o(s) tipo(s) de um, dos 905 Pokémon, e veja qual será sua maior pontuação nas diferentes dificuldades! Por padrão, há 3 tentativas por jogo.

Dificuldades

Há três diferentes dificuldades: fácil, normal (padrão) e díficil.

  • Fácil: Deve acertar pelo menos um tipo.
  • Normal: Deve acertar ambos os tipos, quando houver.
  • Difícil: Deve acertar ambos os tipos, quando houver. Com excessão do Game Over, as tentativas não reiniciam a cada novo jogo, é somado +1 a cada acerto.

Pontuação

Cada dificuldade possui sua pontuação.

  • Acertos: É contabilizado a cada acerto.
  • Melhor: No Game Over, se a quantidade de acertos for melhor que a sua anterior, seus acertos serão atribuídos a ele.

Atenção! Se você mudar a dificuldade com uma quantidade de acertos maior que 0, antes de um Game Over, elas serão zeradas.
